# Network Task 02

## White Band Associates Summer Internship 2026

### Overview

This repository contains the submission for Task 02 of the White Band Associates Summer Internship 2026. The objective of this task was to understand networking fundamentals, network devices, IP address classification, DNS functionality, network communication flow, and practical networking commands in both Windows and Kali Linux environments.

## Task Objectives

* Understand the purpose and functionality of common network devices.
* Differentiate between Public and Private IP addresses.
* Analyze local network configuration details.
* Understand the role of DNS in network communication.
* Visualize the network communication process through a network diagram.
* Perform practical networking commands in Windows and Kali Linux.
